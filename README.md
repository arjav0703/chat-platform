# Full stack Chat Platform
This repository is split into two subparts. Namely Client and Server.

## Usage
1. Clone the repository
```bash
git clone https://github.com/arjav0703/chat-platform.git && cd chat-platform
```

2. Make sure rust (cargo) is installed. If not, visit [here](https://rustup.rs/) to install rust toolchain.
3. Run the server:
```bash
cd server
cargo run
```
4. Make sure pnpm is installed. If not, visit [here](https://pnpm.io/installation) to install pnpm. Then install Tauri prerequisites:
```bash
5. Run the client:
```bash
cd client 
pnpm install
pnpm tauri dev
```
